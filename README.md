# GIS Automation with Python

Automated geospatial workflows using Python to replace manual GIS processes.

## Features
- Vector analysis (buffer and area calculation)
- Raster processing (NDVI generation)
- Spatial joins

## Technologies
- Python
- GeoPandas
- Shapely
- Rasterio
- PostGIS (future integration)

## How to Run
```bash
pip install -r requirements.txt
python scripts/vector_analysis.py
